# Скрипт импорта объявлений с kstu.kz

Консольная команда для скачивания объявлений о защите докторских диссертаций
с сайта [kstu.kz](https://www.kstu.kz/category/obyavleniya-o-zashhite-doktorskoj-dissertatsii/)
и сохранения их в базу данных.

---

## Требования

- PHP 8.0+
- Composer-зависимости установлены (`composer install`)
- База данных создана и миграции применены (`php yii migrate`)
- Доступ в интернет с машины, на которой запускается скрипт

---

## Запуск

Открыть командную строку (cmd или PowerShell) в папке проекта:

```cmd
cd C:\Projects\phd_feed
```

### Базовый запуск — скачать все объявления, сохранить только ссылки на файлы

```cmd
php yii scraper/run
```

### Скачать файлы (PDF и документы) на сервер

```cmd
php yii scraper/run --files=download
```

Файлы сохраняются в `web/uploads/documents/`.

### Выбор языка страницы

```cmd
php yii scraper/run --lang=en
php yii scraper/run --lang=kz
php yii scraper/run --lang=all
```

Если не задано, используется `ru`.

### Только первые N страниц категории

```cmd
php yii scraper/run --pages=3
```

Удобно для тестирования или частичного обновления.

### Все параметры вместе

```cmd
php yii scraper/run --pages=5 --files=download --delay=3
```

---

## Параметры

| Параметр | Значения | По умолчанию | Описание |
|---|---|---|---|
| `--files` | `links` / `download` | `links` | `links` — сохраняет оригинальные URL документов; `download` — скачивает файлы на сервер |
| `--pages` | число | `0` (все) | Количество страниц категории для обработки. `0` = пройти все страницы до конца |
| `--lang` | `ru`, `kz`, `en`, `all` | `ru` | Язык страниц KSTU. `all` обрабатывает все три языка по очереди |
| `--delay` | число секунд | `2` | Пауза между HTTP-запросами. Не ставьте меньше 1 — иначе сайт может заблокировать |

Краткие псевдонимы: `-f`, `-p`, `-d`:

```cmd
php yii scraper/run -f download -p 5 -d 3
```

## Логика работы

```
Для каждой страницы категории:
  → собрать ссылки на объявления

Для каждого объявления:
  → если записи нет в БД → создать новую (статус: опубликовано)
  → если запись есть:
      → сравнить контент (заголовок + текст) с тем что в БД
      → если изменился → обновить, записать updated_at = текущее время
      → если не изменился → пропустить (skipped)

Для документов в объявлении:
  → если --files=links  → записать оригинальный URL в file_path
  → если --files=download → скачать файл, сохранить в web/uploads/documents/
  → дубликаты не добавляются
```

## Что сохраняется в базу данных

| Поле | Откуда берётся |
|---|---|
| `title` | Заголовок `<h1>` страницы |
| `content` | HTML-содержимое `.entry-content` |
| `defense_date` | Ищется дата в тексте (например: «2 июля 2026 года в 10:00») |
| `contact_email` | Первый email-адрес в тексте |
| `zoom_link` | Ссылка zoom.us в тексте |
| `zoom_conference_id` | ID конференции рядом с ключевым словом «ID» |
| `zoom_access_code` | Пароль рядом со словом «пароль» / «код доступа» |
| `url` | Slug из URL страницы (например: `obyavlenie-o-zashhite-ivanov`) |
| `language` | Всегда `ru` (скрапер работает с русской версией) |
| `status` | `published` для всех скачанных объявлений |
| `created_by` | ID пользователя `admin` (id=1) |
| `updated_at` | Обновляется автоматически при изменении записи |

Документы сохраняются в таблицу `announcement_documents`.

## Автоматический запуск по расписанию (Windows)

Через **Планировщик задач Windows** (Task Scheduler):

1. Открыть: `Пуск → Планировщик заданий`
2. `Создать простую задачу...`
3. Имя: `KarTU Scraper`
4. Триггер: `Ежедневно` → выбрать время (например, 02:00)
5. Действие: `Запустить программу`
   - Программа: `C:\php\php.exe` *(путь к вашему php.exe)*
   - Аргументы: `yii scraper/run --files=links`
   - Папка: `C:\Projects\phd_feed`
6. Готово

Проверить путь к PHP:
```cmd
where php
```

## Возможные проблемы

| Проблема | Решение |
|---|---|
| `Не удалось получить страницу` | Проверьте интернет-соединение и доступность kstu.kz |
| `Ошибка сохранения: url already exists` | URL-slug уже занят — возможно, запись создавалась вручную с таким же slug |
| Объявления не находятся (найдено: 0) | Сайт мог изменить HTML-структуру. Сообщите разработчику для обновления парсера |
| Документы не скачиваются | Файлы на kstu.kz могут требовать авторизации — используйте `--files=links` |
